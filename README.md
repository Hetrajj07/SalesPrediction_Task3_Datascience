# Sales Prediction Dashboard

A polished Streamlit app for Oasis Infobyte Task 5.

## Features
- Black themed UI with an upgraded layout
- Interactive sales prediction using `Advertising.csv`
- Model performance metrics and data exploration
- Plotly visualizations for campaign spend and relationships

## Run locally

```bash
cd "/Users/hetrajrana/Desktop/Task 3"
streamlit run app.py
```

If dependencies are missing, install them first:

```bash
pip install -r requirements.txt
```
